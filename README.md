# HomeLibrary
Простенькая программа домашней библиотеки.

Описание проекта:
Данный проект представляет собой консольное приложение, разработанное на языке C#, предназначенное для работы со списком книг домашней библиотеки.

Версии программы:
Версия 1.0:

- Реализован вывод списка книг
- Базовая структура программы
- Отсутствие пользовательского взаимодействия

Версия 2.0:

- Добавлена возможность добавления новой книги
- Реализована проверка корректности ввода
- Улучшен пользовательский интерфейс (цвета, заголовок, форматирование)
- Добавлена нумерация списка книг

Функциональные возможности:

- Отображение списка книг
- Добавление новой книги пользователем
- Обновление и вывод изменённого списка
- Проверка корректности ввода данных
- Цветовое оформление интерфейса

Структура проекта:

- Program.cs — основной файл программы, содержащий всю логику приложения

Используемые технологии:

- Язык программирования: C#
- Среда разработки: Microsoft Visual Studio
- Система контроля версий: Git

Инструкция по запуску:

1. Открыть проект в Visual Studio
2. Запустить программу (кнопка Start или клавиша F5)
3. Следовать инструкциям в консоли
